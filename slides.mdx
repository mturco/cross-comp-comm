## Prop Drilling

```jsx
<Grandparent foo="World" />

// which renders
<Parent foo={foo} />

// which renders
<Child foo={foo} />

// which renders
<p>Hello, {foo}</p>
```

## Context

```jsx
const FooContext = React.createContext();

const Grandparent = () => (
  <FooContext.Provider value="Hello world">
    <Parent />
  </FooContext.Provider>
);
```

---

## Context (cont)

```jsx
const Child = () => {
  const foo = useContext(FooContext);
  return <p>{foo}</p>;
};
```
